# Python Learning Projects

Learning Python through projects organized by chapter.

## Repository Structure

- **ch 13/** - Chapter 13 projects
- **ch 14/** - Chapter 14 projects (Classes and Methods)
- **ch 15/** - Chapter 15 projects (Inheritance and Polymorphism)
- **games/** - Game projects
  - goblin_assault/ - Goblin Assault game

## Getting Started

Each chapter folder contains `.py` files demonstrating concepts from that chapter.

To run any file:
```bash
python3 <filename>.py
```
